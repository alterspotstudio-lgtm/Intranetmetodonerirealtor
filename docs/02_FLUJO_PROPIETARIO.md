# 02 FLUJO PROPIETARIO

## Inicio

El flujo propietario inicia cuando existe acuerdo firmado. Antes de eso es lead.

## Flujo

Landing -> Lead en intranet -> Ficha -> Contacto -> Firma -> Propiedad -> Portal de Venta -> Expediente propietario -> Produccion -> Landing propiedad -> Interesados -> Apartado -> Notaria -> Firma -> Entrega.

## Consolidado

- Lead entra desde landing.
- Ficha muestra datos y primer paso.
- Intranet prepara mensaje.
- Firma permite continuar desde Propiedades.
- Portal de Venta se envia por WhatsApp.
- Expediente propietario abre desde el portal.
- Cliente puede proponer cita de produccion.

## Pendiente

- Link automatico de confirmacion de cita.
- Boton siguiente paso enviar cita.
- Corregir Cita pendiente cuando falta evento o link.
- Cierre formal de entrega.

## No romper

No cambiar esta vista al construir vista comprador.
