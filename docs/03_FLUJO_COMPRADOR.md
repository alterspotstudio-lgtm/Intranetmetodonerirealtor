# 03 FLUJO COMPRADOR

## Inicio

El flujo comprador inicia despues de apartado validado. Antes de eso es lead o interesado.

## Flujo esperado

Lead comprador -> Cita -> Apartado validado -> Portal comprador -> Expediente comprador -> Promesa -> Notaria -> Firma -> Entrega -> Postcierre.

## Consolidado

- Lead comprador llega desde landing.
- Llega vinculado a folio de propiedad.
- Tiene diagnostico, siguiente paso y mensaje sugerido.
- El portal progreso ya tiene base de fases comprador.

## Pendiente

- Candado de apartado con evidencia.
- Link propio de comprador.
- Expediente documental comprador.
- Promesa para ambas partes.
- Confirmacion de notaria.
- Entrega formal.

## Regla

No reutilizar acciones del propietario para comprador sin definir documentos y lenguaje propios.
