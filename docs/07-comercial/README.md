# 07 · Comercial

## Definición comercial

Método NERI no debe presentarse como una intranet ni como un CRM común. La promesa debe centrarse en orden, seguimiento, experiencia de cliente y control operativo.

## Propuesta de valor

Un sistema operativo inmobiliario que guía al asesor paso a paso, documenta la compraventa y mantiene informado al cliente durante la operación.

## Problema que resuelve

- Leads sin seguimiento.
- Asesores que se saltan pasos.
- Clientes que no saben qué está pasando.
- Documentos dispersos por WhatsApp.
- Operaciones sin trazabilidad.
- Directores sin visibilidad del proceso.
- Propiedades que salen al mercado sin preparación.

## Mensaje corto

Método NERI ordena la compraventa inmobiliaria de principio a fin: leads, propiedades, documentos, citas, producción, progreso del cliente y cierre, todo bajo una misma cadena de trabajo auditable.

## Pitch para inmobiliaria

Tu equipo no necesita otro CRM. Necesita un sistema que les diga qué hacer, cuándo hacerlo y qué falta para avanzar cada operación. Método NERI guía al asesor, protege el proceso y le da al cliente una experiencia clara desde el primer contacto hasta la firma.

## Pitch para asesor independiente

Trabaja con estructura de inmobiliaria grande, aunque seas independiente: leads ordenados, mensajes preparados, expediente documental, portal de avance y una experiencia profesional para vendedor y comprador.

## Tiers sugeridos para diseñar después

No fijar precios finales en este documento sin estrategia financiera. Usar como estructura inicial:

### Inicial

Para asesor independiente.

Incluye:

- Perfil asesor.
- Landings.
- Centro de operaciones.
- Leads compradores y vendedores.
- Riel básico.

### Profesional

Para asesor con operación activa.

Incluye:

- Todo lo inicial.
- Expediente documental.
- Portal de Venta.
- Confirmaciones.
- Propiedades.
- Producción inmobiliaria.

### Inmobiliaria

Para equipos.

Incluye:

- Roles.
- Control por asesor.
- Reportes.
- Auditoría.
- Soporte y configuración.
- Branding por cuenta.

## Reglas comerciales

- No prometer reportes automáticos completos hasta terminar `Reportes.html` con datos vivos.
- No prometer app móvil si no existe.
- No prometer WhatsApp API si solo se usan links `wa.me`.
- No prometer IA operativa si el flujo no está conectado y probado.
- No usar palabras técnicas en publicidad.
- No vender funciones pendientes como terminadas.
