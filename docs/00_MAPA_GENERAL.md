# 00 MAPA GENERAL

Metodo NERI es un sistema operativo inmobiliario para guiar, documentar y auditar la compraventa inmobiliaria.

## Cadena general

Perfil asesor -> Landings -> Leads -> Ficha -> Riel de accion -> Firma -> Propiedad -> Portal de Venta -> Expediente propietario -> Produccion inmobiliaria -> Landing propiedad -> Leads compradores -> Cita -> Apartado -> Portal comprador -> Expediente comprador -> Promesa -> Notaria -> Firma -> Entrega.

## Funciona segun prueba viva

- Perfil asesor alimenta intranet y landings.
- Landing vendedor registra lead.
- Ficha vendedor muestra datos y primer paso.
- Riel guia contacto y firma.
- Firma activa continuidad desde Propiedades.
- Portal de Venta se envia al propietario.
- Expediente documental propietario abre desde Portal de Venta.
- Cliente propone cita de produccion.
- Asesor recibe aviso y mensaje preparado.
- Landing propiedad queda lista con folio.
- Leads compradores llegan vinculados a propiedad.

## Pendiente

- Link automatico de confirmacion de cita.
- Boton siguiente paso enviar cita.
- Corregir Cita pendiente en Portal Progreso.
- Mensaje sugerido con envio real.
- Candado de apartado con evidencia.
- Portal comprador propio.
- Expediente documental comprador.
- Promesa, notaria y entrega formal.
